# REPOS Point Card — Android App

A native Android virtual version of the supplied REPOS stamp card.

## Features
- 20 stamp circles in the same 4 × 5 arrangement
- Tap the next empty circle to add a present stamp
- Tap the latest filled circle to undo
- 15th stamp: ¥500 OFF celebration
- 20th stamp: ¥1,000 OFF celebration
- Stamp progress persists after closing the app
- Original card photograph retained inside the app as a branding/artwork reference
- Business hours, closed day, address, phone number, and expiry date included

## Build
Open this folder in Android Studio, allow Gradle to sync, then:
Build > Build APK(s)

The APK will be created under:
app/build/outputs/apk/debug/app-debug.apk
